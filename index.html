• 물류비/보관비: 모델 코드(예: G242L850)로 매칭. 매출이 없어도 물류비/보관비가 있는 제품은 별도 행으로 표시됩니다. (부가세 포함 ×1.1)

매출 데이터

엑셀 헤더 예시: 브랜드, 매출구분, 거래처, 상품코드, 자체코드, 공인바코드, 상품명, 규격, 수량, 평균매출단가, 매출액, 평균공급단가, 공급액, 평균수수료, 수수료, 수수료율, 평균판매원가, 판매원가, 매출이익, 이익율

선택 시 모든 행의 일자가 해당 월(1일)로 저장됩니다. 비워두면 엑셀의 일자 컬럼을 사용합니다.

광고비 데이터

엑셀 헤더 예시: 캠페인, 월별, 총 전환수, 총비용, 직접전환수, 간접전환수, 총 전환매출, 직접전환매출, 간접전환매출, 총 광고수익률, 구매완료 전환수, 구매완료 전환매출액(원)

물류비 데이터

엑셀 헤더 예시: No, 집하일자, 주문유형, 배송처, 배송유형, 채널, F/C명, 운송장번호, 묶음배송번호, 주문번호, 정산분류, 박스구분, 대표품명, SKU건수, 중품목수량, 일반품목수량, 예외요율품목수량, 구성품수량, 추가포장수량, 출고금액, 도선운임, 제주운임, 기타운임, 서비스운임, 정산금액, SBS박스구분, 부자재금액

장려금 데이터

엑셀 헤더 예시: 출고일자, 거래처명, 제품명, 출고수, 원가(출고가), 정산가, 택배 정산, 비고, 장려금 (장려금 열이 실제 장려금 값입니다)

보관비 데이터

엑셀 헤더 예시: No, 마감일자, F/C명, 랙구분, ZONE코드, 로케이션번호, 품목코드, LOT번호, 입고일자, 재고수량, 정산수량, 할증전금액, 보관일수, 할증적용비율, 정산금액 (정산금액 열이 보관비 합산 기준)

ADVoost 광고비 (수동 입력)
